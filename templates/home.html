Welcome to the Stars Track Site!

This is an implementation for two different star tracking algorithms.


Stars Matching Algorithm:

This algorithm locates stars in images, converts them into image-relative points, and generates triangles using Delaunay triangulation. It enables efficient representation and analysis of star positions within the image.

Star Name Algorithm:

The Star Match Algorithm connects to an external database to retrieve images and coordinates. It performs star pattern solving to name each star and determine its location. It then converts image coordinates to WCS (World Coordinate System) coordinates for accurate star positioning.
